# alteraciones en las PFH
En esta presentación se hace un resumen de las _alteraciones de las pruebas de función hepática_, que muchas veces es un hallazgo casual en un paciente asintomático.

La presentación inicial se hizo en el **CS Astillero** en Junio-2026,  por *Socorro Mora*-R3.

Para ello se ha usado el **paquete Xaringan** desarrollado por Yihui Xie.
